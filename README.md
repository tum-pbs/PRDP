# PRDP
Progressively Refined Differentiable Physics

## Requirements
- Python 3.6
- Jax
- Jaxopt
- Equinox
- Optax
- Numpy
- Matplotlib
- ipykernel
- pdequinox
    - ```cd ./pdequinox && pip install -e .```

## How to run

Experiments are implemented as jupyter notebooks in the folder `experiments/`. 

List of experiments:
- 'navier_stokes.ipynb'
